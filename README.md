# Appicoders Website

## Tech Stack

- React.js 18
- Material-UI (MUI) 5
- JavaScript (ES6+)
- CSS-in-JS with emotion



## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
